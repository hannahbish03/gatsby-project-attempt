Fluffy's Carrot Farm - LEGO Microgame Add-ons
========================================
This Microgame Add-On requires: "LEGO Microgame" Version 2.1.0*

You can find more information about this Microgame Add-On here: http://u3d.as/2thb

---------
*: The LEGO Microgame can be obtained from the Unity Hub

REQUIREMENTS
# Unity Hub 2.4.0 or greater; 
# Unity Editor 2020 LTS *exclusively* -- This Microgame is only supported on 2020 LTS

INSTALLATION INSTRUCTIONS
Hub > Projects > (using the "Arrow" key next to "New", select:) 2020 LTS > (in the Templates pop-up window) scroll down to "LEGO Microgame" > select "Download" > then click the tile to launch the LEGO Microgame
